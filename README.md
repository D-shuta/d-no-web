# 特徴  

* サイトに動画を乗せる事でSEO効果が５３倍！  
* 検索ページの１ページ目に表示されやすくする効果がある  
* 今回、テンプレートで参考にした
[ペライチ](https://peraichi.com/)
ではトップページに動画が乗っていない   
* 有名なサイトに対して、後追いでも動画を活用すれば有名なサイトに負けない可能性あり！
